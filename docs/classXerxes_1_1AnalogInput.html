Detailed Description

Analog input sensor class.

AnalogInput is a generic class for up to 4 analog inputs, e.g. 4x 0-3.3V

Note
default ADC depth on RP2040 is 12 bit, 8.7 ENOB (approx. 54dB SNR)
AnalogInput uses oversampling to increase resolution, increasing SNR by 6dB per bit
n-bit oversampling increases sampling time too: sample time = 4^n * conversion time
