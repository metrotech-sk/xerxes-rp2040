◆ readRegCallback()

void Xerxes::readRegCallback ( const Xerxes::Message &  msg)

Read register callback.

Read <LEN> bytes from device register, starting at <REG_ID> The request prototype is <MSGID_READ> <REG_ID> <LEN>

Parameters
msg
Note
This function is blocking, it will not return until the data is read from the register and sent to the master device.
All data are in little endian format - LSB first.
